# HTML Elements Supported by Watir

Watir can access following HTML elements (alphabetically):

    <area>
    <a>
    <button>
    <dd>
    <div>
    <dl>
    <dt>
    <em>
    <form>
    <frame>
    <h1>
    <h2>
    <h3>
    <h4>
    <h5>
    <h6>
    <iframe>
    <img>
    <input type="button">
    <input type="checkbox">
    <input type="file">
    <input type="hidden">
    <input type="image">
    <input type="password">
    <input type="radio">
    <input type="reset">
    <input type="submit">
    <input type="text">
    <input>
    <label>
    <li>
    <map>
    <p>
    <pre>
    <select>
    <span>
    <strong>
    <table>
    <tbody>
    <td>
    <textarea>
    <tr>
    <ul>

If that is not enough for you, Watir also has XPath support. You can use it to access any HTML element.
